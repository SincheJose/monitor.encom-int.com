How It Works: Each client site runs a lightweight agent that monitors local Canon printers and sends data to this central dashboard.

Secure

Outbound HTTPS only

Firewall Friendly

No inbound ports needed

Auto-Discovery

Finds printers via SNMP

Windows Installation

canon-agent-installer.exe --api-key YOUR_API_KEY

Linux Installation

curl -sSL https://monitor.encom-int.com/agent.sh | sudo bash

System Requirements

  • • Windows Server 2012+ or Linux Ubuntu 18.04+
  • • 512MB RAM, 100MB disk space
  • • Network access to Canon printers via SNMP
  • • Outbound HTTPS port 443 allowed
